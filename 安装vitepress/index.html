安装vitepress ​

1.在目录中先安装vitepress ​

cmd
pnpm add -D vitepress

可能遇到安装不上的情况 ​

解决方案 ​

cmd
pnpm config set puppeteer_skip_chromium_download true -g

2.初始化vitepress的cli ​

cmd
npx vitepress init

3.根据提示配置项目的一些基本信息 ​

4.在项目根目录下安装依赖包 ​

cmd
pnpm i

5.启动项目 ​

cmd
pnpm run docs:dev
